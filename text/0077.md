# 77 - தமரும் அமரும்





| தனன தனன தனன தனன
| 
|      தனன தனன ...... தனதான
| 
| 
| ......... பாடல் .........
| 
| 
| தமரு மமரு மனையு மினிய
| 
|      தனமு மரசும் ...... அயலாகத்
| 
| 
| தறுகண் மறலி முறுகு கயிறு
| 
|      தலையை வளைய ...... எறியாதே
| 
| 
| கமல விமல மரக தமணி
| 
|      கனக மருவு ...... மிருபாதங்
| 
| 
| கருத அருளி யெனது தனிமை
| 
|      கழிய அறிவு ...... தரவேணும்
| 
| 
| குமர சமர முருக பரம
| 
|      குலவு பழநி ...... மலையோனே
| 
| 
| கொடிய பகடு முடிய முடுகு
| 
|      குறவர் சிறுமி ...... மணவாளா
| 
| 
| அமர ரிடரு மவுண ருடலு
| 
|      மழிய அமர்செய் ...... தருள்வோனே
| 
| 
| அறமு நிறமு மயிலு மயிலு
| 
|      மழகு முடைய ...... பெருமாளே.
