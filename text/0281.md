# 281 - இறையத்தனையோ





| தனனத் தனனா ...... தனதான
| 
| 
| ......... பாடல் .........
| 
| 
| இறையத் தனையோ ...... அதுதானும்
| 
| 
| இலையிட் டுணலேய் ...... தருகாலம்
| 
| 
| அறையிற் பெரிதா ...... மலமாயை

|

| 
| மறையத் தனைமா ...... சிறைசாலை

| 
| வழியுய்த் துயர்வா ...... னுறுதேவர்

| 
| சிறையைத் தவிரா ...... விடும்வேலா
| 
| 
| திலதைப் பதிவாழ் ...... பெருமாளே.
