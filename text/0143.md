# 143 - வரிக் கலையின்

| வரிக்கலையி னிகரான விழிக்கடையி லிளைஞோரை
| 
|      மயக்கியிடு மடவார்கள் ...... மயலாலே

| 
| ஒருத்தருட னுறவாகி ஒருத்தரொடு பகையாகி

|      ஒருத்தர்தமை மிகநாடி ...... யவரோடே

|      திருக்கையினில் வடிவேலை ...... யுடையோனே

|      திருத்தணிகை மலைமேவு ...... பெருமாளே.
