# 354 - கருப்பற்று ஊறி





| தனத்தத் தானத் ...... தனதான
| 
| 
| ......... பாடல் .........
| 
| 
| கருப்பற் றூறிப் ...... பிறவாதே
| 
| 
| கனக்கப் பாடுற் ...... றுழலாதே
| 
| 
| திருப்பொற் பாதத் ...... தநுபூதி
| 
| 
| சிறக்கப் பாலித் ...... தருள்வாயே
| 
| 
| பரப்பற் றாருக் ...... குரியோனே
| 
| 
| பரத்தப் பாலுக் ...... கணியோனே
| 
| 
| திருக்கைச் சேவற் ...... கொடியோனே
| 
| 
| செகத்திற் சோதிப் ...... பெருமாளே.
