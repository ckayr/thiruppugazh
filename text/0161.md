# 161 - அதிரும் கழல்

| 
|      தனனந் தனன தந்த ...... தனதான
| 
| 
| ......... பாடல் .........

| 
| அதிருங் கழல்ப ணிந்து ...... னடியேனுன்

|      அபயம் புகுவ தென்று ...... நிலைகாண

| இதயந் தனிலி ருந்து ...... க்ருபையாகி

| பதியெங் கிலுமி ருந்து ...... விளையாடிப்

|      பலகுன் றிலும மர்ந்த ...... பெருமாளே.
