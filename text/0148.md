# 148 - பத்தியால் யானுனை





| தத்தனா தானனத் ...... தனதான
| 
|      தத்தனா தானனத் ...... தனதான
| 
| 
| ......... பாடல் .........
| 
| 
| பத்தியால் யானுனைப் ...... பலகாலும்
| 
|      பற்றியே மாதிருப் ...... புகழ்பாடி

| 
| முத்தனா மாறெனைப் ...... பெருவாழ்வின்

| 
| உத்தமா தானசற் ...... குணர்நேயா

| 
| 
| வித்தகா ஞானசத் ...... திநிபாதா
| 
|      வெற்றிவே லாயுதப் ...... பெருமாளே.
