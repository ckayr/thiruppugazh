# 9 - சந்ததம் பந்த





| தந்தனந் தந்தத் ...... தனதான
| 
|      தந்தனந் தந்தத் ...... தனதான
| 
| 
| ......... பாடல் .........
| 
| 
| சந்ததம் பந்தத் ...... தொடராலே
| 
|      சஞ்சலந் துஞ்சித் ...... திரியாதே
| 
| 
| கந்தனென் றென்றுற் ...... றுனைநாளும்

|      சங்கரன் பங்கிற் ...... சிவைபாலா
| 
| 
| செந்திலங் கண்டிக் ...... கதிர்வேலா
| 
|      தென்பரங் குன்றிற் ...... பெருமாளே.
