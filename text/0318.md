# 318 - உரத்துறை போத





| தனத்தன தானத் ...... தனதான
| 
| 
| ......... பாடல் .........
| 
| 
| உரத்துறை போதத் ...... தனியான
| 
| 
| உனைச்சிறி தோதத் ...... தெரியாது
| 
| 
| மரத்துறை போலுற் ...... றடியேனும்
| 
| 
| மலத்திருள் மூடிக் ...... கெடலாமோ
| 
| 
| பரத்துறை சீலத் ...... தவர்வாழ்வே
| 
| 
| பணித்தடி வாழ்வுற் ...... றருள்வோனே
| 
| 
| வரத்துறை நீதர்க் ...... கொருசேயே
| 
| 
| வயித்திய நாதப் ...... பெருமாளே.
